Vim on iPadOS 中文文档 
=======

Vim on iPadOS 中文文档计划

# 关 于

 是一个功能非常强大，且具有很强扩展性的编辑器。而且
Vim 本身带有一个完备的帮助系统。iVim 是在 iOS 和 iPadOS 上运行的 Vim。

本项目的目的就是将 Vim 和 iVim 的这些文档翻译成中文，以便更多的人认识及更好地使用这个非常强大的编辑器。

文档分成用户手册和参考手册两部分，你既可以像使用教程那样循序渐进，也可以快速地查阅来获取帮助。
